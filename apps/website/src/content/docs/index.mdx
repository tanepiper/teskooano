## More Features

<CardGrid>
  <Card title="Intuitive Controls" icon="setting">
    Select, focus and orbit around any celestial body with intuitive camera
    controls, orbit tracking, and detailed information panels.
  </Card>
  <Card title="Offline-First PWA" icon="phone">
    Responsive and mobile-friendly, Teskooano can be fully installed on any
    device as a Progressive Web App that can be used offline
  </Card>
  <Card title="Dynamic Time Control" icon="rocket">
    Adjust simulation speed from real-time to 10 million times faster to observe
    both immediate interactions and long-term celestial evolution.
  </Card>
  <Card title="Modular Architecture" icon="approve-check-circle">
    Built with TypeScript and modern frontend tools, Teskooano features a
    component-based design that's easily extendable.
  </Card>
</CardGrid>

## Key Capabilities

<CardGrid>
  <Card title="Accurate Celestial Physics" icon="puzzle">
    Full N-Body gravitational simulation where every object exerts gravitational
    force on every other object. Watch planets orbit stars, moons orbit planets,
    and complex gravitational interactions create realistic celestial motion.
  </Card>
  <Card title="Real-time Visualization" icon="seti:lua">
    Built on Three.js, Teskooano renders complex celestial systems in real-time
    with physically accurate lighting, materials, and effects supporting
    hundreds of objects while maintaining smooth performance.
  </Card>
  <Card title="Interactive Learning Platform" icon="open-book">
    Perfect for educational settings, making complex orbital mechanics intuitive
    and accessible. Experiment with different planetary configurations and learn
    about celestial mechanics through direct observation.
  </Card>
  <Card title="Seamless User Experience" icon="rocket">
    An interactive guided tour introduces you to all key features. The intuitive
    DockView-based UI allows for customizable layouts to suit your specific
    needs.
  </Card>
  <Card title="Open Technology Stack" icon="laptop">
    Built with Three.js for 3D rendering, TypeScript for type-safe development,
    Nanostores for state management, Driver.js for tours, and DockView for
    flexible layouts.
  </Card>
  <Card title="Open Source & Community Driven" icon="github">
    Completely open source under the MIT license. We welcome contributions, bug
    reports, and feature requests from the community via{" "}
    <a href="https://github.com/tanepiper/teskooano" target="_blank">
      GitHub
    </a>
  </Card>
</CardGrid>
